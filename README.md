Project title-Creating Our Own VIRTUAL ASSISTANt

Description:

This project is an AI-powered virtual assistant developed using Python and machine learning techniques. The assistant can recognize voice commands, process user requests, and perform tasks such as opening websites, searching Wikipedia, telling the current time, sending emails, and playing music. The system uses speech recognition and Natural Language Processing (NLP) concepts for intelligent command understanding and response generation.


Features:

Voice command recognition
Text-to-speech response
Wikipedia search
Open websites like YouTube and Google
Tell current time
Send emails using voice commands
Play music
Intent recognition using Machine Learning
Error handling for invalid commands
User-friendly voice interaction


Technologies Used:

Python
SpeechRecognition
Pyttsx3
Scikit-learn
TF-IDF Vectorizer
Naive Bayes Algorithm
Pandas
NumPy
Wikipedia API
SMTP
WebBrowser Module
